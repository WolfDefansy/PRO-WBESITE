Welcome on board

Main page for recruitment and info

Wolfdefansy is an association for high schoolers worldwide, offering support, free learning, technical challenges, and a friendly atmosphere, all on a free CTF platform. Our mission is to make cybersecurity accessible to everyone, regardless of background or experience. We believe in sharing knowledge, helping each other, and building a strong, inclusive community. Join us to discover, learn, and grow in cybersecurity, with no barriers or limits.

WolfDefansy makes cybersecurity learning accessible and simple for all.

Our mission is to guide students with a complete, centralized path, offering support, resources, and a welcoming community for everyone interested in learning.

We need you!

WolfDeFansy is looking for contributors for its website. You don't need to be elite, but basic HTML, CSS, and TypeScript/JavaScript are required.

The project uses Ionic (multi-platform) and Angular (good code separation). Basics are enough.

Prerequisites

  • Know basic git commands (repo on GitHub)
  • Some object-oriented programming knowledge
  • Can style a component from a mockup (HTML & CSS)
  • Enjoy teamwork (sharing, communication, good mood)

Motivated beginners are welcome. The team and Wiki will help you. Learning is collaborative.

Topics covered

  • Component creation
  • Service creation (sync/async)
  • Data with RxJS or Angular Signals
  • User event handling
  • Dynamic display with Angular
  • And more!

You don't need to know everything. The key is knowing how to search: docs, Google, or Copilot.

Our team is supportive. Motivation and involvement are the keys to progress.

HTML 5: The skeleton of websites

Learn HTML

HTML is the foundation of every website. It’s not a programming language, but a markup language that lets you structure your content: titles, paragraphs, lists, images, and links.
With HTML, you build the backbone of your pages, giving meaning and order to your information. Mastering HTML is the first step to understanding how the web works and is essential before learning CSS or JavaScript. It’s the universal language for web content, and knowing it lets you read, edit, and create any web page.

Discover CSS

CSS is what brings style and life to your web pages. It lets you control colors, fonts, spacing, and even animations, separating content from presentation.
With CSS, you can create beautiful layouts, responsive designs, and smooth transitions. It adapts your site to all screens—computers, tablets, and phones. CSS is the creative layer that makes your HTML unique and user-friendly, allowing you to design modern, attractive, and accessible interfaces for everyone.

Learn JavaScript

JavaScript is the language that makes your web pages interactive and dynamic. It adds features like events, dynamic content, and communication with servers.
With JavaScript, you can create menus, forms, games, and real-time updates.
